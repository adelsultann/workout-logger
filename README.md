OverloadPro is a Flutter app built for lifters who follow the **progressive overload** principle.  
No more paper logs or messy spreadsheets—just clear data and smart nudges.

| Feature | Details |
|---------|---------|
| **Set‑by‑Set Logging** | Reps, weight, RPE, rest—captured in seconds |
| **Automatic Overload Suggestions** | Algorithm flags when to add weight or reps |
| **Visual Progress Charts** | Strength curves per lift, muscle group, or block |
| **Custom Routines** | Push/Pull/Legs, 5×5, or your own plan |
| **Offline Friendly** | Sqflite local storage; sync later if you create account |
| **Anonymous Sign‑In** | Start training instantly—no email required |
| **Data Export** | One‑tap CSV for coaches or backup |

### Tech Stack
Flutter • Dart • Sqflite • Google TTS (optional cues) • Firebase Auth (anonymous) • Provider / Riverpod
